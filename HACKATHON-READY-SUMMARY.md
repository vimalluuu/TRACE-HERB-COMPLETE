## 🔧 **TROUBLESHOOTING**

### **If Ports Are Busy**
```bash
# Kill processes on specific ports
taskkill /F /PID <PID>
# Or restart computer
```

### **If Node.js Missing**
- Install from https://nodejs.org/
- Ensure it's added to PATH
- Restart command prompt

### **If Dependencies Fail**
```bash
# Clear cache and retry
npm cache clean --force
```
